FAQ

Do I need an API key?
No. Sawarou does not use any AI API. It's a local editor — you load HTML generated by ChatGPT or Claude and edit it entirely on your own PC.
Does it work with any HTML file?
It works with HTML generated by Claude, ChatGPT, or other AI tools, as well as hand-written HTML. It does not support WordPress, Shopify, or framework-based files (React, Vue, etc.).
Does it work on Mac and Windows?
Yes. As long as you have Google Chrome on Mac or Windows, it will work.
Can I publish the edited HTML directly?
Yes. The downloaded HTML is clean and ready to upload to any server. Editor code is automatically removed.
